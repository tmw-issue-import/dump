  • Cybeloras posted a comment Aug 5, 2012

    Seeing as you are using an alpha version, your probably not a new user of TellMeWhen, but I will offer these basic suggestions anyway, just in case: If there are no groups shown in config mode, this could indicate a few things:

    1. None of your groups are set to show for your current talent tree (Balance, Feral, Restoration)
    2. None of your groups are set to show for your current talent specialization (Primary/Secondary)
    3. Your groups have been moved off your screen (unlikely, but possible.)

    All of the above issues can be solved by typing '/tmw options' and changing these settings for your current groups, or by adding new groups.

    What is the much more likely cause of the error, however is that there is a Lua error occurring that is preventing the proper execution of the code that handles TMW's configuration mode. Make sure Lua errors are enabled in Blizzard's interface options (under the Help sub-category) and post any errors that pop up that pertain to TellMeWhen.

    If you already have Lua errors enabled, however, and no errors are occurring, then the best way for me to diagnose this issue would be to have you upload your SavedVariables file. Saved variables can be found in: "World of Warcraft/WTF/Account/<Account Name>/SavedVariables/TellMeWhen.lua". You can paste the contents of the file to http://wow.curseforge.com/paste/ and link to it here, or user another method to provide me with the file.

    Looking forward to getting your issue resolved! Thank you for using TellMeWhen!
