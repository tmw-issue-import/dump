    What version of TellMeWhen are you using? ("The latest" is not a version)

    7.0.2

    Please provide any additional information below (including any export strings if applicable; see above.)

    I want to make TellMeWhen icons able to check a specific raid frame, regardless of its raid ID. For example, I want to make TMW check the first raid frame for my rejuvenation with a standard buff/debuff tracker, no matter what that players raid unit ID may be. I use VuhDo for raid frames, so I can sort them by unit ID, and then it should work by having the first rejuvenation icon check raid 1, the second check raid 2, etc. However, I like to sort them by tank > healer > DPS, which means my icon for raid 1 would be on the first VuhDo raid frame, but it would still be checking raid 1, who could be anywhere. I assume this is possible via Lua, and I'm trying to learn Lua, but sadly I don't have the faintest idea how to make this work.

    Anyone reading this may be confused as to why I want to do this, since VuhDo can already show rejuvenation on frames. As great as VuhDo is, it doesn't have even a fraction of the customization that TMW offers in icons. With TMW I can control precisely when, where, and how information shows up on the raid frames. I can add conditions, notifications, and there are tons of interesting applications with DogTags text customization. Then there's just the fact that TMW is more responsive and easier on my computer's CPU.

    Anyway, I'm not exactly sure what this ticket is, so I'm really sorry if this doesn't belong here or is formatted wrong. I'm not sure if this is a question of how to do something or a suggestion of a feature to be implemented. Either way, thanks for making and maintaining this awesome addon! In my opinion it is, by far, the best WoW addon ever created.

    Then,

    • Manually run the snippet, or reload your UI to have it automatically run.
    • Name a group with a name that begins with "VuhDo" (no quotes, case insensitive)
    • Anchor that group to a VuhDo frame (right-click-and-drag the group to a vuhdo frame and choose the "anchor to" option from the dropdown menu that appears.)
    • That group's icons will automatically be updated to that frame's unit.
