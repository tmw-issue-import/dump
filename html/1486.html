BATTLE FOR AZEROTH VERSIONS #1486

  • Other
Closed
  • Cybeloras created this issue Apr 20, 2018
    Owner

    If you're looking for BFA-compatible versions of TMW, look no further.

     

    Unfortunately, with all the features that Curse/Twitch/Amazon has removed from Curseforge, including multiple repository forks each with their own file sets, it won't be easy like it was in the past. Here's how to to get the latest:

     

    1. Exit WoW if its running.
    2. Download and install a live version of TMW into your BFA Alpha WoW installation. Use the latest alpha version listed on https://wow.curseforge.com/projects/tellmewhen/files .
    3. Download https://github.com/ascott18/TellMeWhen/archive/bfa.zip, extract the contents (the files and folders inside "TellMeWhen-bfa"), and copy them right on top of their counterparts in the live version you just installed, choosing to replace all files with the same names.
    4. Open Interface/Addons/TellMeWhen/TellMeWhen_Options and copy TellMeWhen_Options.toc into Interface/Addons/TellMeWhen_Options/TellMeWhen_Options.toc
    5. If you want non-English translations, keep the files from live in the TellMeWhen/Localization folder (basically, undo the replacement for these files only)

     There are probably lots of things broken right now - please consolidate all issues with the BFA versions into comments on this ticket (preferably complete with stack traces, please).
