  • Cybeloras posted a comment Jun 24, 2013

    They are indeed related to TellMeWhen, although there is nothing that I can do about them - the error is in Blizzard's code. There is some issue with the texture Interface\Buttons\UI-SquareButton-Disabled.tga (which baffles me because TMW does not use that texture anywhere). That is the only thing that all these errors have in common - some of them are caused when TellMeWhen is in the process of calling a Blizzard API function (so far I have seen it happen with GetSpellInfo, GetItemInfo, and GetGlyphInfo), but some others are happening when TMW is not running any code at all.

    I guess we will just have to wait it out until Blizzard fixes things. I can load up the PTR without issue after enduring about 5 of these crashes. That is a sure sign that this is something on Blizzard's end of things (in the client code) and not TMW. Addons should never, under any circumstance, be able to completely crash the game. Any time that they do is a flaw in Blizzard's code.

    Thank you for reporting this. Make sure to also report the error to Blizzard when the dialog pops up.
