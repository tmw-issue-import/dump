    ^1^T^SType^Scleu ^SSettingsPerView^T ^Sicon^T ^STexts^T ^N1^S[Source] ^t^t^t^SDestUnit^Splayer ^SCLEUEvents^T ^SSPELL_AURA_APPLIED^B ^t^SEnabled^B ^t^N60101^S`| ^Sicon^^

    This will check for any buffs applied to you and type the name of the source in the icon for 5 seconds. Copy paste it into the import text field, then press "Import/Export/Restore", then select "From String", then selected "((Unnamed))".
