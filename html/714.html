  • Cybeloras posted a comment Nov 28, 2012

    Neither the Relic of Yu'lon nor the Mithril Wristwatch are trinkets that have On Use effects. The description for the Icon Cooldown icon type clearly states that it only tracks the cooldowns of items with Use effects.

    You will need to use the Internal Cooldown icon type in order to track the cooldowns of these two trinkets. Please be sure to read all of the tooltips that you see as you set this up - especially the tooltip for the Internal Cooldown icon type, as well as the tooltips for the Triggered By settings, and most importantly the text contained in the big yellow popups that may appear.
