    This is a longstanding error with the cooldown animations on Blizzard's side. Occasionally, even if their frame level is explicitly set to be on top of the icon texture by TMW, they will sometimes go underneath. OmniCC's text will still appear, but the cooldown animation will not. Ive tried to fix this in about a dozen different ways, but it still seems to slip through sometimes. You should be able to fix it by toggling config on and off (/tmw).
