    IMPORTANT: If your issue is only occurring with a specific icon, include an export string of the icon in this ticket. Subsequently, if the issue involves an entire group or an entire profile, include the entire export string of the group or profile.

    To get an export string, open the icon editor, and click on the arrow next to the editbox labeled "Import/Export/Backup". Select the "To string" option for the appropriate export type (icon, group, or profile), and then press CTRL+C to copy it to your clipboard.

    If you are experiencing errors that seem to be unrelated to icons, attaching a saved variables file will increase the likelihood of your problem being solved.     2. Check the use box for Counting, Timer Shown, Always Show at the color settings for any item, white color.
    3.

    version 6.0.1

    the problem doesn't produce any kind of error messages
